Detailed Description

Instructs Spring MVC on how to parse and print elements of type 'PetType'. Starting from Spring 3.0, Formatters have come as an improvement in comparison to legacy PropertyEditors. See the following links for more details: - The Spring ref doc: https://docs.spring.io/spring-framework/docs/current/spring-framework-reference/core.html#format
